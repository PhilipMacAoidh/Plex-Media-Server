HOW DOES IT WORK


Well... It's based on an application called 'Plex Media Server', that's what I use to display everything nicely including the posters, cast, trailers. All of that is done with Plex. I then use 2 different applications to keep the TVShows and Movies downloaded and up to date. They're called Sonarr (TV) and Radarr (Movies).

Sonarr creates a release calendar for itself so that it can search torrent sites (more on that later) for new episodes of TV Shows that I want to keep updated. Sonarr can also download past episodes that I missed, but it's not reliable in any capacity. Sonarr can also download the video quality of your choice. I have it set to download 720p videos but if none can be found in that quality it'll download them in 1080p, if they're available. Once it finds the episode it sends the link to BitTorrent and then downloads and organises it into the correct show and season folders, and renames the file so Plex can find the file and understand what episode it is.

Radarr allows me to request movies. It'll wait until the movies been digitally released to start searching the torrent sites. Just like Sonarr, Radarr can download movies in my selected quality, I have it set to search for 1080p videos but if they're unavailable then it'll search for 720p. It can download movies that have been previously released, but it can be very hit and miss depending on how popular the movies is and how long ago it was released. Once it finds the movie, it sends it onto BitTorrent, downloads it, renames it, and organises it into the correct folder for Plex.

I use another application called Jackett. Sonarr and Radarr have some torrent sites that they can search but it's not reliable or maintained. Jackett can search ALL the sites. They just need to be configured and off you pop.

The website is run off a NGINX server. This basically sees that someone wishes to connect to the server and depending on the url (e.g. /help, /about, /plex ...) it will route the user to the correct page.
